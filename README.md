## 📌 Project Overview

This project is an interactive Ecommerce Sales Dashboard created using Microsoft Excel.  
The dashboard helps analyze:

- 💰 Total Sales
- 📈 Profit Trends
- 🛒 Quantity Sold
- 🌍 Regional Performance
- 📊 Category-wise Insights

---

## 🚀 Features

✅ KPI Cards  
✅ Interactive Slicers  
✅ Sales Trend Analysis  
✅ Profit Analysis  
✅ Region-wise Insights  
✅ Category Performance Tracking  

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
